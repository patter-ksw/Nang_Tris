# 프로젝트 개요: 고양이 테트리스 "냥트리스" (Antigravity 로우코드 개발)

개인 포털 'Youns-pg'에 링크로 연동할 고양이 테마의 테트리스 게임 "냥트리스"를 개발하고자 합니다. 
개발 툴은 **Antigravity(안티그래비티)**를 사용할 예정이며, 데이터베이스 및 인증은 **Supabase**를 활용합니다. 
싱글 플레이 및 실시간 1:1 멀티플레이 모드와 메달이 표시되는 랭킹 보드를 지원해야 합니다. 안티그래비티의 노코드/로우코드 컴포넌트 구조와 이벤트 핸들러(워크플로우)에 맞게 가이드를 작성해 주세요.

## 1. 데이터베이스 구성 및 SQL (Supabase)
안티그래비티의 Supabase Integration 연동을 위한 테이블 구조와 SQL DDL 스크립트를 제공해 주세요.
1. **기존 사용자 테이블:** `tr_users` (유저 ID 고유값 참조)
2. **순위 저장 테이블:** `tr_rankings` (싱글/멀티 최고 점수, 달성일 저장. `tr_users` 테이블의 ID를 외래키로 참조)

## 2. Antigravity 페이지 및 컴포넌트 트리 구조 (UI 레이아웃)
안티그래비티 빌더화면에서 시각적으로 배치할 수 있도록 페이지별 컴포넌트 계층 구조를 제안해 주세요.
- **메인 로비 페이지 (`/lobby`):**
  - 싱글 플레이 진입 버튼
  - 멀티플레이 방 만들기 컴포넌트 (모달창: 방 제목 입력, 비밀번호 On/Off 스위치, 생성 버튼)
  - 개설된 방 리스트 뷰 (비밀번호 방은 잠금 아이콘 표시)
- **랭킹 보드 컴포넌트:**
  - 사용자의 개인 최고 기록 카드 (Supabase 데이터 바인딩)
  - 전체 순위 테이블 뷰: Supabase에서 `Score` 기준 내림차순(DESC) 정렬로 데이터를 바인딩합니다.
  - **메달 조건부 렌더링:** 랭킹 인덱스가 1등이면 금메달(🥇), 2등은 은메달(🥈), 3등은 동메달(🥉) 컴포넌트/아이콘을 보여주고, 4등부터는 일반 숫자가 보이도록 안티그래비티 내 조건부 노출 설정을 알려주세요.
- **게임 플레이 페이지 (`/game`):**
  - 싱글 모드 화면: 내 Canvas 컴포넌트 + 점수/Next 블록 UI
  - 멀티 모드 화면: 내 Canvas(좌) + 상대방 실시간 화면 미러링 Canvas(우)

## 3. 핵심 비주얼 및 사운드 연출 ("냥트리스" 테마)
- **고양이 미노(Mino):** 4개의 셀이 연결된 블록을 귀엽고, 뚱뚱하고, 날씬한 고양이 캐릭터 스프라이트 이미지로 렌더링할 수 있도록 자바스크립트 커스텀 스크립트 블록이나 Canvas 드로잉 설계를 해주세요.
- **스테이지 클리어 애니메이션:** 게임 승리나 스테이지 클리어 이벤트가 발생하면 춤추는 이쁜 고양이 캐릭터 컴포넌트(GIF 또는 Lottie 애니메이션 소스 활용)가 화면에 오버레이 팝업으로 나타나며 신나는 BGM이 나오도록 연출 구조를 잡아주세요.
- **사운드 효과:** 줄 제거 시 '야옹~' 소리 등 효과음 오디오 파일 재생 액션 연동.

## 4. Antigravity 워크플로우 및 스크립트 요구사항
- **[1단계]:** Supabase에 생성할 랭킹 테이블 SQL 스크립트를 작성해 주세요.
- **[2단계]:** 안티그래비티에서 Supabase 데이터를 불러와 1~3등에게 금/은/동메달을 매핑해 뿌려주는 **랭킹 리스트 데이터 바인딩 로직**과, 싱글/멀티 게임판의 기본 레이아웃을 구성하기 위한 **커스텀 컴포넌트 가이드**를 작성해 주세요.
